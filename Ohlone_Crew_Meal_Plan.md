## Emergency Contacts & Safety

| Contact | Number | Notes |
|---|---|---|
| **East Bay Regional Parks Dispatch** | **510-881-1121** | 24/7 — primary for Ohlone Trail incidents |
| **Alameda County Sheriff SAR** | **510-667-7721** | Search & rescue if park staff unavailable |

| Hazard | Notes |
|---|---|
| **Cell coverage** | Intermittent — move to higher ridgeline to improve signal. Note GPS coords at each camp. |
| **Poison oak** | Common along trail edges all 3 days. "Leaves of three, let it be." Wear long sleeves on brushy sections. |
| **Ticks** | Common in spring grass. Tuck pants into socks. Full-body tick check every evening before bed. |
| **Cattle** | Active grazing on Day 2 ranchland. Stay calm, walk slowly around herds, give wide berth. |
| **Sun exposure** | Long exposed ridgelines all 3 days. Hats, sunscreen, sun shirts mandatory. |
| **Permits** | Required for **all** Ohlone Wilderness campsites. Carry confirmation — rangers check. |
